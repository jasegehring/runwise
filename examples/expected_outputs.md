# Runwise Example Outputs

This document shows example outputs from Runwise commands to help you understand what to expect.

## List Runs

```bash
runwise list
```

**Output:**
```

ID           State     Date            Steps     Accuracy        Trend
------------------------------------------------------------------------

def456       CRASHED   2025-12-13      2,341       45.0%    ▁▁▂▅▇▇▇▇↑
```
